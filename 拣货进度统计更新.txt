【 技术实现 】

统计方式：
- 遍历已走过的路径
- 筛选出 action === 'pickup' 的步骤
- 计数得到已拣货物数量

代码逻辑：
```javascript
if (gameState.isRunning) {
    // 运行中：统计已拣货物数量
    aiPickedCount = gameState.aiPath
        .slice(0, gameState.aiVehicle.pathIndex)
        .filter(p => p.action === 'pickup').length;
    
    playerPickedCount = gameState.playerPath
        .slice(0, gameState.playerVehicle.pathIndex)
        .filter(p => p.action === 'pickup').length;
}
```

【 与其他统计的关系 】

拣货进度 vs 总步数：
- 拣货进度：3/10（已拣3件）
- 总步数：45（已走45步）
→ 平均每件货物走 45÷3 = 15步

拣货进度 vs 当前载重：
- 拣货进度：3/10
- 当前载重：120kg
→ 平均每件货物 120÷3 = 40kg

拣货进度 vs 卸货次数：
- 拣货进度：7/10
- 卸货次数：2
→ 拣了7件货物，卸了2次货
